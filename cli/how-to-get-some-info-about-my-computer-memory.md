You can use free command:

```
$ free -h
total       used       free     shared    buffers     cached
Mem:          7,7G       7,3G       443M       348M       349M       1,9G
-/+ buffers/cache:       5,0G       2,6G
Swap:         3,9G       303M       3,6G
```
